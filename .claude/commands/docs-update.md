# 문서 업데이트 스킬

OpenSpec 문서와 CLAUDE.md를 최신 상태로 유지합니다.

## 지시사항
1. 코드 변경사항을 확인 (git diff, 수정된 파일)
2. 관련 문서 업데이트:
   - CLAUDE.md — 프로젝트 개요/구조 변경 시
   - docs/specs/api-reference.md — API 변경 시
   - docs/specs/database-schema.md — 스키마 변경 시
   - docs/specs/project-overview.md — 아키텍처 변경 시
3. 문서 간 일관성 확인
4. 너무 큰 문서는 분할

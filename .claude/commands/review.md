# 코드 리뷰 스킬

프로젝트 코드를 리뷰하고 개선점을 도출합니다.

## 지시사항
1. 보안 취약점 검사 (XSS, SQL injection, CSRF 등)
2. 코드 품질 검사 (중복, 복잡도, 에러 핸들링)
3. 성능 이슈 확인
4. 발견된 이슈를 심각도별 분류:
   - Critical: 즉시 수정 필요
   - High: 빠른 수정 권장
   - Medium: 개선 권장
   - Low: 참고 사항
5. Critical 이슈는 GitHub 이슈로 등록
